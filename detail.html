2024年国家公务员考试真题（行测）

📅 考试年份：2024年 📝 题目数量：135题 👁️ 浏览次数：19.9万 ⏰ 更新时间：2024-12-01
单选题 第 1 题
下列关于我国宪法修改的表述，正确的是（）。
A 宪法修改权由全国人民代表大会行使
B 宪法修改须由全国人大常委会或者五分之一以上的全国人大代表提议
C 宪法修改须由全国人大全体代表的三分之二以上的多数通过
D 以上都对
正确答案： D

答案解析

根据我国宪法规定，宪法的修改，由全国人民代表大会常务委员会或者五分之一以上的全国人民代表大会代表提议，并由全国人民代表大会以全体代表的三分之二以上的多数通过。因此，A、B、C三项均正确，答案为D。

单选题 第 2 题
根据《民法典》的规定，下列关于自然人民事权利能力的表述，正确的是（）。
A 自然人从出生时起到死亡时止，具有民事权利能力
B 自然人的民事权利能力一律平等
C 胎儿不具有民事权利能力
D 死者不具有任何民事权利
正确答案： B

答案解析

根据《民法典》第14条规定，自然人的民事权利能力一律平等。第13条规定，自然人从出生时起到死亡时止，具有民事权利能力，依法享有民事权利，承担民事义务。第16条规定，涉及遗产继承、接受赠与等胎儿利益保护的，胎儿视为具有民事权利能力。但是，胎儿娩出时为死体的，其民事权利能力自始不存在。故B选项正确。

单选题 第 3 题
下列关于行政行为的表述，正确的是（）。
A 行政行为都是单方行为
B 行政行为都是要式行为
C 行政行为都是自由裁量行为
D 行政行为是行政主体行使行政职权的行为
正确答案： D

答案解析

行政行为是指行政主体行使行政职权，作出的能够产生行政法律效果的行为。A项错误，行政行为既有单方行为也有双方行为（如行政合同）；B项错误，行政行为既有要式行为也有非要式行为；C项错误，行政行为既有自由裁量行为也有羁束行为；D项正确，行政行为必须是行政主体行使行政职权的行为。

单选题 第 4 题
下列关于刑法中因果关系的表述，正确的是（）。
A 因果关系是犯罪构成的必备要件
B 因果关系是行为人承担刑事责任的充分条件
C 因果关系是行为与结果之间的客观联系
D 因果关系只存在于故意犯罪中
正确答案： C

答案解析

因果关系是指危害行为与危害结果之间的引起与被引起的客观联系。A项错误，因果关系只是结果犯的必备要件，行为犯、危险犯不需要因果关系；B项错误，因果关系是行为人承担刑事责任的必要条件而非充分条件；C项正确，因果关系是行为与结果之间的客观联系；D项错误，因果关系既存在于故意犯罪中，也存在于过失犯罪中。

单选题 第 5 题
下列关于我国人民代表大会制度的表述，错误的是（）。
A 人民代表大会制度是我国的根本政治制度
B 人民代表大会制度实行民主集中制原则
C 全国人民代表大会是最高国家权力机关
D 地方各级人民代表大会受上级人民代表大会的领导
正确答案： D

答案解析

根据我国宪法规定，地方各级人民代表大会是地方国家权力机关，它们与上级人民代表大会之间不存在领导关系，而是监督关系。上级人民代表大会及其常务委员会监督下级人民代表大会及其常务委员会的工作。因此D选项表述错误，当选。
